# SistemaNotasUsuario

## Descripción.

Aplicación de consola en java que permite registrar usuarios, iniciar sesión y gestionar notas de usuarios.
La información la guardamos en ficheros de texto.

## Cómo Ejecutar.

1. Abrir el proyecto en el IDE.
2. Ejecutar el app/Main.java.

## Funcionalidades.

Registro e inicio de sesión, contraseña protegida con SHA-256, crear, listar, ver y eliminar notas, busqueda de notas por palabra clave, guardado en fichero temporal.

## Arquitectura.

-app.
-model.
-service.
-utils.
-view.
